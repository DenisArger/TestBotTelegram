# Memory Bank — TestBotTelegram

Краткое долговременное описание проекта для быстрой ориентации и передачи контекста.

## Назначение

Проект предоставляет минимальную функцию для отправки сообщений в Telegram Bot API. Используется как строительный блок для ботов и уведомлений.

## Текущие возможности

- Отправка текста в чат по `chat_id`.
- Использование `axios` для HTTP запросов.
- Экспорт функции `sendMessage(chat_id, text)`.

## Архитектура

- Однофайловый модуль `sendMessage.js`.
- CommonJS экспорт (`module.exports = async (...) => { ... }`).

## Внешние зависимости

- `axios` — HTTP клиент.
- `@notionhq/client` — пока не используется напрямую, оставлен для будущих интеграций.

## Конфигурация

- `BOT_TOKEN` (env) — токен Telegram-бота. Обязателен для работы.

## Точки расширения

- Дополнительные параметры Bot API (`parse_mode`, `disable_web_page_preview`).
- Другие методы API: `sendPhoto`, `sendDocument`, `editMessageText`.
- Интеграция с Notion для генерации уведомлений.
- Retry/backoff, логирование и трейсинг ошибок.

## Ограничения/риски

- Нет валидации входных параметров.
- Ошибки `axios` не перехватываются внутри функции.
- Нет тестов.
